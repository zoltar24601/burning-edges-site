🎯 FanDuel Slate Upload

Upload a FanDuel slate CSV to use FD salaries and positions in EDGE. Toggle to FD mode in the main app after uploading.

How to get the CSV:

  1. Go to FanDuel MLB and enter (or open) any contest on tonight's slate
  2. From the contest page → Edit Lineups → Download CSV
  3. File is named like FanDuel-MLB-2026-XX-XX-XXXXX-entries-upload-template.csv
  4. Drop it into the box below

This CSV contains both your active contest entries (left side) AND the full player pool (right side). EDGE parses both — entries get preserved for round-trip lineup edits, players become your FD salary pool.
